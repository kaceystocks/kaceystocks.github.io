HTML Basics

🧱The Structure of the Web

What is HTML? HTML (HyperText Markup Language) is the skeleton of every website. It defines the meaning and structure of web content. We use tags to create elements, which the browser then displays.

HTML Tags, Elements, and Attributes

A tag looks like <p> or </p>

An element is the opening tag, the content, and the closing tag (e.g. <p>This is content.</p>)

Attributes provide extra information about the element (e.g. <img scr="image.png"/>)

Tags

Tags are the primary way you mark up content in HTML. They are keywords enclosed in angle brackets (< and >).

  • Structure: Most tags come in pairs: an opening tag and a closing tag.
  • Purpose: Tags tell the web browser what kind of content is inside them.

Opening: <h1> Closing: </h1>

The img tag is one of the few self-closing tags.

<img>

Elements

An element is the complete unit formed by the opening tag, the closing tag, and all the content in between.

  • Structure: Opening Tag + Content + Closing Tag = Element
  • Purpose: An element defines a specific piece of the document structure (like a complete title, paragraph, or list item).

<p>This is a complete paragraph element.</p>

Some tags, like the image tag or the line break tag, don't need content and do not have a separate closing tag. They are a complete element by themselves.

<img>, <br>

Attributes

Attributes provide extra information about an element. They are always placed inside the opening tag and are defined as name/value pairs.

  • Structure: name="value"
  • Purpose: Attributes modify the element or provide necessary details the browser needs to process it.

Example 1 (Styling): The class attribute assigns a name for styling with CSS.

<h1 class="main-title">My Heading</h1>

Example 2 (Functionality): The src attribute is essential for the img tag, telling the browser the source (location) of the image file.

<img src="picture.jpg" alt="A descriptive label">

Here, src and alt are the attribute names, and "picture.jpg" and "A descriptive label" are their values.
