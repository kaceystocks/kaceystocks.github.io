What is HTML? HTML (HyperText Markup Language) is the skeleton of every website. It defines the meaning and structure of web content. We use tags to create elements, which the browser then displays.

HTML Tags, Elements, and Attributes

A tag looks like <p> or </p>

An element is the opening tag, the content, and the closing tag (e.g. <p>This is content.</p>)

Attributes provide extra information about the element (e.g. <img scr="image.png"/>)
