What is JavaScript? JavaScript (JS) is the action layer of the web. While HTML provides structure and CSS provides style, JavaScript makes your website interactive and dynamic. It allows you to respond to user actions (like button clicks), manipulate data, and change HTML and CSS in real-time.

Variables, Data Types, and Functions

Variables: are a container for storing data values.

Data Types: are a classification that specifies the kind of value a variable can hold.

Functions: are blocks of code designed to perform a particular task. They help organize your code and make it reusable.
