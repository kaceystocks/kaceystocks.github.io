🧠Logic and Interactivity

What is JavaScript? JavaScript (JS) is the action layer of the web. While HTML provides structure and CSS provides style, JavaScript makes your website interactive and dynamic. It allows you to respond to user actions (like button clicks), manipulate data, and change HTML and CSS in real-time.

Variables, Data Types, and Functions

Variables: are a container for storing data values.

Data Types: are a classification that specifies the kind of value a variable can hold.

Functions: are blocks of code designed to perform a particular task. They help organize your code and make it reusable.

Variables

Declaration Keywords:

  • var : Older keyword, generally avoided in modern JS due to scoping issues.
  • let : Used when you need to reassign the variable's value later.
  • const : Used for values that should not change after they are set.

Example:

let age = 30;

Data Types:

  • String - "Hello World" - Textual data, enclosed in quotes.
  • Number - 10, 3.14, -5 - Numeric data (integers and floating-point numbers).
  • Boolean - true or false - Represents a logical entity (yes/no).
  • Null - null - Represents the intentional absence of any object value.
  • Object - {key: value} - Used for complex data structures (e.g., arrays, functions, and objects).

Example:

const greeting = "Hi"; // String

Functions

Defining a function:

function greet(name) { return "Hello" + name + "!"; };

Function Expression (often assigned to a const variable):

const multiply = function(a, b) { return a * b; };

Calling a function:

let message = greet("Charlie"); // message is now "Hello, Charlie!"

let product = multiply(5, 3); // product is 15
