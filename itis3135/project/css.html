CSS Styling

🎨 Styling the Web

What is CSS? CSS (Cascading Style Sheets) is the "skin" or stylist for your HTML structure. It controls the visual appearance of your content—things like colors, fonts, layouts, and spacing. CSS separates presentation from structure, making your code cleaner and easier to manage.

Selectors

A Selector is the part of the CSS rule that identifies which HTML element(s) the rules should apply to. It's how you target specific tags, classes, or IDs.

Selector Type Example What it Targets Explanation
Element/Type p All <p> tags Selects all instances of a specific HTML tag.
Class .warning All elements with class="warning" Allows you to apply the same style to many different elements across the page.
ID #header The single element with id="header". IDs must be unique on a page, making this the most specific type of selector.
Universal * Every element on the page. Selects everything. Often used for resetting styles.

Declarations

A CSS declaration is the core component that defines how a specific style rule should be applied to an HTML element. It's the part that tells the browser what change to make and how to make it.

Property

The Property is the human-readable identifier that tells the browser what characteristic of the element you want to change.

Examples:

color: Changes the text color.

font-size: Changes the size of the text.

margin: Controls the spacing outside the element's border.

Value

The Value is the specific setting or configuration you want to apply to the property. It tells the browser how to apply the change.

Examples:

blue (for the color property).

10px 20px (for the font-size property).

10px or 30% (for the margin property).

Box Model

Every element on a web page is treated as a box composed of four distinct layers:

  • Content: This is the innermost part of the box, containing the actual text, images, or other media. Its size is determined by the element's width and height properties.
  • Padding: This is the transparent space that immediately surrounds the content. It acts as a buffer zone between the content and the border.
  • Border: This is a line that wraps around the padding and the content. It is the visible edge of the box.
  • Margin: This is the outermost transparent space that surrounds the border. It is used to create space between the current box and other adjacent boxes on the page.
